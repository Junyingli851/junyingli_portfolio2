Tutor Orial

[ Project Statement]

Tutor Orial is an online tutoring service that connects high school and college students to fun and friendly tutors who can help them learn. The main goal for this project was to enhance the sign-up flow as they have a very low completion rate. My role in this project was a UX designer.This is my first try to make user flow and interview people to learn real user needs. How to filter a key point from a lot of feedback for me was a big challenge! i learn a lot from this project.

[ Problem Statement ]

Problem Statement: How might we increase Tutor orial's conversion rate, specifically on the tutor searching page and the student scheduling screen?

[ Solution ]

Redesign the website flow, find a tutor, and personal learning data.
